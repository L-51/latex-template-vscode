# Additional Information
Here are some useful links.

## LaTeX Documentation and Tutorials
### [LaTeX Workshop Documentation](https://github.com/James-Yu/LaTeX-Workshop)
This is the repository and documentation of the author of the extension used by this project.
> Additional useful information.

### [Simple LaTeX Manual](https://manualdelatex.com/tutoriales)
A tutorial to learn how to write in LaTeX, with very good documentation and broad coverage for beginners. Available in Spanish.
> Very useful for learning LaTeX commands.

### [Basic Markdown Writing](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)
Tutorial on how to write in **Markdown**, very useful since **GitHub** repositories use Markdown for documentation.  
> Simple introduction to Markdown writing.

### [LosDelGIIM Templates](https://github.com/LosDelDGIIM/LosDelDGIIM.github.io/tree/main/subjects/plantillas)
Examples of templates used for note-taking projects, exam uploads, course material, etc.  
> Useful as reference for practical LaTeX documents.
